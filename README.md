# PowerBI-Dashboard-Project
This project is a Sales Analytics Dashboard created for a Store using Power BI. The dashboard provides a clear and interactive view of business performance including total sales, profit, quantity sold, category-wise analysis, and regional insights.
<img width="1941" height="1072" alt="PowerBI Dashboard Project" src="https://github.com/user-attachments/assets/238c2ded-d837-469d-b247-4529d1cf1907" />

Key Features:

Interactive Sales Dashboard

KPI Cards (Sales, Profit, Quantity)

Category & Sub-category Analysis

Monthly Sales Trend Visualization

Region-wise Performance Insights

User-friendly filters and slicers

Objective:
To transform raw sales data into meaningful business insights that help in better decision-making and performance tracking.
